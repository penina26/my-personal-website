# Pesh-AI: Data Science & Data Engineering Portfolio

<p>A lightweight, single-page portfolio built with vanilla <strong>HTML, CSS, and JavaScript</strong> to showcase projects, skills, and contact info for Penina Wanyama.</p>

<span><strong>|</strong></span> <em>Live site:</em> https://penina26.github.io/my-personal-website/ <br>

## Features
<ul>
    <li>Responsive layout with a project grid and mobile nav (hamburger, expandable menu)</li>
    <li>Scrollspy navigation: active menu item highlights as you scroll through sections</li>
    <li>Back-to-top button with smooth scrolling back to the hero section</li>
    <li>Accessible contact form with inline validation, ARIA alerts, and error highlighting</li>
    <li>SEO-ready: descriptive <code>&lt;title&gt;</code>, meta description/keywords, canonical URL, Open Graph &amp; Twitter cards</li>
    <li>Clean UI elements: tags/chips, animated hover underline, dark theme</li>
    <li>Zero dependencies: no JS frameworks required</li>
    <li>Tiny JS script: mobile nav toggle, scrollspy, back-to-top behavior, dynamic footer year, and client-side form validation</li>
</ul>

## Tech Stack
<ul>
    <li><strong>HTML5</strong> for structure (semantic sections for hero, projects, about, contact)</li>
    <li><strong>CSS3</strong> for layout and styling (Flexbox, Grid, custom properties / variables, responsive breakpoints)</li>
    <li><strong>Vanilla JavaScript</strong> for navigation toggle, scrollspy, smooth scrolling, and form validation</li>
    <li><strong>Font Awesome</strong> for navigation, call-to-action, and education icons</li>
    <li><strong>GitHub Pages</strong> for hosting and deployment</li>
</ul>
